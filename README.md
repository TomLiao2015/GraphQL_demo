### Prepare
- Install mongodb
- Initial DB
```
    npm run initDB
```

### Start server
```angular2html
    npm install
    npm run start
```

You can access below url to test
- GraphQL Doc to testing GraphQL api
```angular2html
    http://localhost:4000/demo/graphiql
```
- Swager doc to testing rest api
```angular2html
    http://localhost:4000/rest/demo/
```
